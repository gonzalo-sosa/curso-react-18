# CSS en línea

CSS en línea es el CSS que se incluye directamente en el atributo "style" en el componente.

```tsx
function App() {
  return (
    <div style={{ color: "red" }}>
      Container
    </div>
  );
}
```

El atributo "style" espera un objeto con propiedades CSS y valores (por lo general) en formato string o number.
